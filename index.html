Predicting Maintenance for manufacturing

This project focuses on predicting the Remaining Useful Life (RUL) of machinery using sensor data. Utilizing the NASA CMAPS dataset,the project involves comprehensive data preprocessing,feature engineering, and model building. Various regression models,including Random Forest and Gradient Boosting,were trained and evaluated. Hyperparameter tuning using RandomizedSearchCV was employed to optimize the model performance. This showcases my ability to handle real-world predictive maintenance tasks.
